## 1. 네비게이션 구조

하단 탭 바 4개:

| 탭 | 할일\|스케줄 탭 | 설명 |
|---|---|---|
| inbox | ✅ | 미분류 item + envelope 없는 paper |
| papers | ✅ | Envelope별 paper 관리 |
| schedule | — | 전사 날짜별 스케줄 뷰 |

## 2. Inbox 화면

### 2-1. 전체 구조

- 상단: 페이지 타이틀 "inbox" + 우상단 활성/완료 토글
- 콘텐츠 영역: draft card (최상단 고정) + envelope 없는 paper card 목록
- 최하단: 할 일 입력창 고정 (탭 바 바로 위)

### 2-2. 활성/완료 토글

- 위치: 우상단
- 스타일: 작은 pill 형태 텍스트 토글 (버튼 스타일 아님)
- **활성 탭**: draft card + 활성 paper 목록
- **완료 탭**: 완료된 paper 목록만 (draft card 없음)

### 2-3. 할 일 입력창

- 위치: 최하단 고정 (탭 바 바로 위)
- 입력 시 draft card 최상단에 미리보기로 흐릿하게 표시
- 전송 시 draft card 최상단에 항목 추가

### 2-3-1. 날짜 태그 (선택사항)

- item 추가 시 날짜 태그 선택 가능 (선택 안 해도 됨)
- 옵션: 오늘 / 내일 / 일주일 / 날짜 직접 선택
- 태그된 item은 inbox 상단 캘린더 바에 표시

### 2-3-2. Paper 선택 (선택사항)

- item 추가 시 대상 paper 선택 가능 (미선택 시 draft로)
- 선택 가능 목록: inbox 내 active paper 목록
- InputBar에 paper 이름 chip 형태로 표시

### 2-3-2. 주간 캘린더 바

- 위치: inbox 상단 (draft card 위)
- 이번 주 날짜를 가로 스크롤로 표시
- 날짜별로 scheduled item 수 표시 (점 또는 숫자)
- 특정 날 탭하면 해당 날의 item만 필터링
- 날짜 없는 item은 항상 표시

### 2-4. Draft Card

- 항상 최상단 고정, 순서 조정 불가
- 스타일: 점선 테두리 + 배경색으로 paper와 구분
- 구성: draft 라벨 + 정렬 선택(생성일↓↑/마감일↓↑) + N/M 카운트 + progress bar + item 목록
- 항목 정렬: 기본 생성일↓ (미체크 상단, 체크 하단 + 타임스탬프)
- 새 항목 추가 위치: **최상단** (가장 최근 항목이 위로)
- 항목 10개 초과 시: 처음 10개만 표시, "N개 더보기" / "접기" 토글
- **완료 버튼**: 항목이 1개 이상일 때 노출. 완료 시 → draft items를 오늘날짜 paper로 묶어 완료 처리 + wave 생성

### 2-5. Envelope 없는 Paper Card (활성)

- 접힌 상태 기본 (papers 화면과 동일한 UX)
- 꾹 누르면 바텀시트 → envelope 선택하여 이동

### 2-6. Item/Paper 분류 (바텀시트)

- item 또는 paper 꾹 누르면 바텀시트 등장
- 바텀시트: "어느 envelope로 분류할까요?" + envelope 목록 (색상 dot + 이름 + paper 수)
- envelope 선택 시 해당 envelope의 draft paper로 편입
- item 우측 분류 아이콘(⊹) 상시 노출로 기능 발견성 확보

### 2-7. 완료 탭

- 완료된 paper 목록만 표시 (draft card 없음)
- paper 구성: paper 이름 + 완료 시각 + item 목록
- item 표시: 체크 항목 상단 + 타임스탬프 / 미체크 항목 하단 (텍스트 스타일 변경 없음)

## 3. Papers 화면

### 3-1. 전체 구조

- 상단: 페이지 타이틀 "papers" + 우상단 활성/완료 토글
- Envelope 탭: 가로 스크롤 pill 형태. 우측 끝에 + 버튼 (envelope 추가)
- Envelope 간 좌우 스와이프로 이동 가능
- 탭 하단 구분선
- 콘텐츠 영역: draft card (최상단 고정) + paper card 목록

### 3-2. 활성/완료 토글

- 위치: 우상단
- 스타일: 작은 pill 형태 텍스트 토글 (버튼 스타일 아님)
- 옵션: 활성 / 완료

### 3-3. Draft Card

- 항상 최상단 고정, 순서 조정 불가
- 스타일: 점선 테두리 + 배경색으로 일반 paper와 구분
- 구성: draft 라벨 + 정렬 선택(생성일↓↑/마감일↓↑) + N/M 카운트 + progress bar + item 목록 + 항목 추가 버튼
- 항목 10개 초과 시: 처음 10개만 표시, "N개 더보기" / "접기" 토글
- **완료 버튼**: draft paper가 존재하고 항목이 있을 때 노출. 완료 시 → draft paper name을 오늘날짜(YYYY-MM-DD)로 변경 후 완료 처리 + wave 생성

### 3-4. Paper Card — 접힌 상태 (기본)

- 구성: paper 이름 + 우측 N/M 카운트 + 꺽쇠(›) + progress bar
- 탭하면 펼쳐짐

### 3-5. Paper Card — 펼친 상태

- 스타일: 초록 테두리로 선택 상태 표시
- 구성:
  - 상단: paper 이름 + 접기 버튼(∨)
  - progress bar
  - item 목록 (미체크 상단, 체크 하단)
  - 항목 추가 버튼 ("+ 항목 추가...")
  - 하단 footer: progress bar + N/M 카운트 + 완료 버튼
- 완료 버튼: 초록 배경 액션 버튼 스타일
- 모든 item 체크 여부와 관계없이 완료 처리 가능

### 3-6. Item 추가 위치 정책

| 구분 | 새 항목 추가 위치 |
|---|---|
| draft card | **최상단** (최근 항목이 위) |
| 일반 paper | **최하단** |
| 공통 | 드래그앤드롭으로 순서 조정 가능 |

### 3-7. Paper Card — Item 표시 규칙 (활성)

- 미체크 항목: 상단 정렬, 일반 텍스트
- 체크된 항목: 하단 정렬, 취소선 + 흐린 폰트색 + 타임스탬프 (오전 8:02)

### 3-7-1. InputBar (papers)

- 위치: 최하단 고정 (탭 바 바로 위)
- 날짜 태그 선택 가능 (오늘/내일/일주일/직접 선택)
- Paper 선택 가능: 현재 envelope 내 active named paper 목록 chip 표시. 미선택 시 draft로

### 3-8. Paper 추가 버튼

- paper 목록 하단에 두 버튼 나란히, 동일 스타일 (점선 테두리)
- **새 paper**: 빈 paper 생성
- **새 wave**: 즐겨찾기 목록 바텀시트 → 선택 시 복제해서 새 활성 paper 생성

### 3-9. 새 wave 바텀시트

- 즐겨찾기된 paper 목록 (이름 + 항목 미리보기 + wave 수)
- 하단에 "전체 완료 paper에서 검색..." 옵션

### 3-10. Paper 순서 조정

- 별도 관리 화면에서 조정 (상세 기획 추후)

## 4. Papers 화면 — 완료 탭

- Draft card 없음
- Paper 추가 버튼 없음
- 항목 추가 버튼 없음

### 4-1. 완료된 Paper Card 구성

- 상단: paper 이름 (좌) + 완료 시각 + ★/☆ (우)
- ★/☆ 위치: 완료 시각 오른쪽 (아이템 타임스탬프와 의도적으로 얼라인 어긋남)
- ★/☆ 탭으로 즐겨찾기 즉시 토글
- Item 목록:
  - 체크된 항목: 상단 정렬, 체크 표시만 (취소선/폰트색 변경 없음) + 타임스탬프
  - 미체크 항목: 하단 정렬, 텍스트 스타일 변경 없음
- 꾹 누르면 바텀시트: 복제해서 시작 / 삭제

## 6. Me 화면

### 6-1. 전체 구조

- 상단: 페이지 타이틀 "me" + 우상단 ⚙ 설정 진입
- 내부 탭: **dashboard | challenges**

### 6-2. Dashboard 탭

#### Streak 카드
- 기준: 앱에서 item을 1개라도 체크한 날 카운트
- 표시: 연속 일수 (크게) + 최장 기록 + 최근 30일 중 사용일

#### 이번 주 카드
- 이번 주 wave 수 + 지난 주 대비 증감
- streak 카드와 나란히 컴팩트하게 배치

#### 연도 네비게이터
- 위치: streak/이번 주 카드 바로 아래
- 좌우 꺽쇠로 연도 이동
- 현재 연도에서 미래 연도 이동 불가

#### 연간 히트맵
- 기준: wave 수
- GitHub 잔디 스타일, 가로 스크롤
- 월 라벨 표시
- 범례: "연간 기록" 라벨과 같은 줄 우측 배치

#### 루틴별 누적 wave
- envelope별 wave 누적 수
- envelope 색상 dot + 이름 + 바 차트 + wave 수

### 6-3. Challenges 탭

- 상단: 전체 달성 진행도 (N/전체 + 프로그레스 바)
- 달성 개수에 따라 레벨업, 레벨업 시 새 챌린지 잠금 해제
- 카테고리별 뱃지 그리드 (4열)
- 잠긴 뱃지: 흐릿하게 표시
- 히든 챌린지: "?"로 표시, 레벨 조건 충족 시 공개

#### 챌린지 카테고리
- 첫 발걸음 (온보딩 관련)
- 꾸준함 (streak 기반: 3/7/30/100/365일)
- wave 마스터 (누적 wave: 10/50/100/500/1000)
- 루틴 전문가 (envelope별 wave 누적)
- 수집가 (다양성: envelope 수, 즐겨찾기 수 등)
- 숨겨진 도전 ✦ (히든: 자정 wave, 새벽 5시 전 체크 등)

## 11. Schedule 화면

### 11-1. 전체 구조

- 상단: 페이지 타이틀 "schedule"
- 캘린더 바: 이번 주 날짜 가로 표시. 날짜별 dot으로 item 수 표시. 탭하면 해당 날짜 섹션으로 스크롤 점프 (필터 아님)
- 본문: 날짜별 그룹 주루룩 스크롤
- 최하단: 할 일 입력창 고정

### 11-2. 캘린더 바

- 오늘 날짜: 초록 강조
- 날짜별 dot: scheduled item 수 표시
- 탭: 해당 날짜 섹션으로 앵커 스크롤 (필터 아님, 전체 날짜 항상 표시)

### 11-3. 날짜별 그룹

- 전사 item 표시 (inbox item + 모든 paper item)
- item 옆에 출처 표시 (inbox / envelope 이름)
- 오늘: 초록 강조
- 지난 날짜: 흐릿하게 + "지남" 표시

### 11-4. Item 생성

- 하단 입력창에서 바로 추가
- paper 지정 선택 가능 (미지정 시 inbox로)
- 날짜 태그 선택 가능

### 11-5. Inbox / Papers 내 스케줄 탭

- inbox와 papers 각 화면에 할일 | 스케줄 탭 존재
- inbox 스케줄: inbox item 한정
- papers 스케줄: 해당 envelope 내 item 한정 (맥락별 관리)
- 구조는 schedule 탭과 동일하되 범위만 다름
